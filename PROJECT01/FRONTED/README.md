# Bharat Safar - Explore India

A travel destination website built using HTML, CSS, and JavaScript.

## Features

* Responsive User Interface
* Travel Destination Cards
* Search Functionality
* Beautiful Travel-Themed Design
* Easy Navigation

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

FRONTED/
├── images/
├── index.html
├── style.css
└── script.js

## How to Run

1. Open the FRONTED folder.
2. Double-click `index.html`.

OR

Use VS Code Live Server.

## Learning Outcomes

* Responsive Web Design
* DOM Manipulation
* JavaScript Event Handling
* Frontend Development Basics
