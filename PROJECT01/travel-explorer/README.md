# BharatSafar Travel API

A RESTful Travel Explorer and Booking Platform built using Node.js and Express.js.

## Features

* Retrieve All Destinations
* Search Destinations
* View Destination Details
* Create Bookings
* Retrieve Bookings
* Contact API
* Error Handling Middleware

## Tech Stack

* Node.js
* Express.js
* JavaScript

## API Endpoints

### Destinations

GET /api/destinations

GET /api/destinations/:id

GET /api/destinations?search=query

### Bookings

GET /api/bookings

POST /api/bookings

### Contact

POST /api/contact

## Installation

Install dependencies:

npm install

Run the server:

npm start

Server URL:

http://localhost:5000
